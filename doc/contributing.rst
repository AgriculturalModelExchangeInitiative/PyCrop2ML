Contributing Guide
^^^^^^^^^^^^^^^^^^^

This is a wiki for anything related to the contributing on `Crop2ML <https://github.com/AgriculturalModelExchangeInitiative>`_ which is a project of the
Agricultural Model Exchange Initiative.
